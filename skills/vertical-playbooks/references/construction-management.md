# Construction Management Companies Playbook

## Overview

**Target Profile:**
- Owner's representatives and construction managers
- Project management consultants
- Program managers overseeing multiple projects
- No direct construction work - oversight and coordination
- Sweet spot: Managing $10M-500M in projects

**Key Decision Makers:**
- Principal/Partner
- Senior Project Manager
- Director of Operations
- Business Development Lead

---

## Top Pain Points

### 1. Multi-Contractor Coordination
**Symptoms:**
- Herding cats across contractors
- Schedule conflicts between trades
- Information silos per contractor
- Owner drowning in updates

**Questions:**
- "How do you coordinate between multiple contractors?"
- "How do you ensure contractors share information?"
- "How do you identify conflicts before they happen?"
- "How do you keep owners informed without overwhelming them?"

**Automation:**
- Multi-contractor status dashboard
- Conflict detection alerts
- Owner digest reports
- Cross-contractor communication hub
- Milestone synchronization

**ROI Calculation:**
```
Hours/week on coordination × Hourly rate × 4.33
Plus: Cost of coordination failures × Frequency
```

---

### 2. Owner Communication Management
**Symptoms:**
- Owners asking for updates constantly
- Report preparation time sink
- Different owners want different formats
- Important issues buried in noise

**Questions:**
- "How do owners get project updates?"
- "How much time goes into owner reporting?"
- "How do you escalate issues to owners?"
- "How do you handle different reporting requirements?"

**Automation:**

**ROI Calculation:**
```
PM hours/week on owner communication × Number of PMs × Hourly rate
```

## CM-Specific Workflows

### Owner Reporting Flow
1. Collect contractor updates (auto-request)
2. Aggregate data by category
3. Generate draft report
4. PM review and edit
5. Distribute to owner
6. Track feedback/questions

### Issue Resolution Flow
1. Issue identified
2. Route to responsible party
3. Track response deadline
4. Escalate if overdue
5. Document resolution
6. Update owner if significant

### Change Order Review Flow
1. Receive contractor CO request
2. Review supporting documentation
3. Validate against contract
4. Recommend to owner
5. Track owner decision
6. Document final outcome
