## How to Use

1. Use Apify Twitter scraper to fetch latest tweets from all handles
2. Extract: tool names, product launches, trending topics, links
3. Look for patterns: multiple influencers discussing the same tool/topic = high signal
4. Flag any heated debates or contrarian takes = potential video content
5. Note engagement levels (likes/retweets) as buzz indicators
