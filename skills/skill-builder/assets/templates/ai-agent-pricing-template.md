# AI Agent Pricing Framework

## Pricing Models

### 1. Per-Agent Monthly Fee
Best for: Predictable workloads, SME clients

| Tier | Agents | Price/mo | Features |
|------|--------|----------|----------|
| Starter | 1 | $297 | Basic automation |
| Growth | 3 | $697 | + Integrations |
| Scale | 10 | $1,997 | + Custom agents |

### 2. Usage-Based
Best for: Variable workloads, enterprise

```
Base fee: $99/mo
+ $0.02 per task executed
+ $0.10 per API call
Cap at: $X/mo (optional)
```

### 3. Value-Based / ROI Share
Best for: High-impact automations

```
Setup fee: $2,500 - $10,000
+ 10-20% of documented savings
Minimum: $500/mo
```

## Pricing Psychology

- Anchor high, offer discount
- Show "hours saved" not just cost
- Use odd numbers ($297 vs $300)
- Bundle for perceived value
- Offer annual discount (15-20%)

## Israeli Market Adjustments

- Quote in ₪ (NIS) for local SMBs
- VAT: +17% (show inclusive price)
- Payment: Credit card or bank transfer
- Common: 3-6 month commitments
