## Content Guidelines

### Blog Post Structure

1. **Title (H1)**: Include primary keyword, 50-60 chars
2. **Introduction**: Hook + keyword in first 100 words
3. **Body**:
   - H2s for main sections (include keywords)
   - H3s for subsections
   - Bullet points for scannability
   - Internal links to related posts
   - External links to authoritative sources
4. **Conclusion**: Summarize + CTA
5. **Word Count**: 1,500-2,500 for pillar content

### On-Page SEO Checklist

- [ ] Primary keyword in title
- [ ] Primary keyword in first paragraph
- [ ] Primary keyword in at least one H2
- [ ] 2-3 related keywords naturally included
- [ ] Meta description with primary keyword (150-160 chars)
- [ ] Alt text on all images
- [ ] Internal links (3-5 per post)
- [ ] External links (1-2 to authoritative sources)
- [ ] URL contains primary keyword (slug)
